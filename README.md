# 🔥 KAI CHEF - מדריך התקנה

**מטרה:** להעלות את האפליקציה לאוויר, לקבל קישור לשלוח לחברים, כל זה בתוך 20-30 דקות מהיום.

---

## 📋 מה תצטרכי לפני שמתחילים

- [ ] כרטיס אשראי (לחשבון API של Anthropic - חינם עד ~$5, אחר כך תשלום לפי שימוש)
- [ ] כתובת מייל
- [ ] דפדפן Chrome או Safari

**עלות משוערת לחודש בטסטים ראשונים:** $2-10 (תלוי בכמות המשתמשים)

---

## שלב 1: הוצאת API Key מ-Anthropic (5 דקות)

1. היכנסי ל: **console.anthropic.com**
2. הרשמי או התחברי
3. הזיני פרטי אשראי (Settings → Billing)
4. עברי ל-**API Keys** → **Create Key**
5. תני שם למפתח (למשל: `kai-chef-production`)
6. **העתיקי את המפתח מיד** - הוא מתחיל ב-`sk-ant-...`
7. **שמרי אותו במקום בטוח** - לא תראי אותו שוב

⚠️ **חשוב:** אל תשלחי את המפתח הזה לאף אחד, אל תעלי אותו ל-GitHub, אל תשמרי אותו בהודעה. הוא כמו סיסמה לכספת שלך.

---

## שלב 2: יצירת חשבון GitHub (5 דקות)

GitHub הוא איפה שנשמור את הקוד.

1. היכנסי ל: **github.com**
2. **Sign up** - הזיני מייל, סיסמה, שם משתמש
3. אשרי את המייל
4. אחרי ההתחברות, לחצי על **+ (למעלה מימין) → New repository**
5. שם: `kai-chef-app`
6. סמני **Public** (זה בסדר - הקוד לא מכיל סודות)
7. סמני **Add a README file**
8. לחצי **Create repository**

---

## שלב 3: העלאת קבצי הפרויקט (5 דקות)

1. בעמוד שנפתח (של ה-repository החדש), לחצי **Add file → Upload files**
2. פתחי את התיקייה שהורדת (`kai-chef-app`)
3. **גררי את כל הקבצים** מהתיקייה אל החלון של GitHub
   - חשוב: לגרור את *התוכן* של התיקייה, לא את התיקייה עצמה
4. חכי שכולם יעלו (יש פס התקדמות למטה)
5. בתחתית העמוד, לחצי **Commit changes**

תראי עכשיו את כל הקבצים בעמוד ה-repository.

---

## שלב 4: העלאה לאוויר עם Vercel (5 דקות)

Vercel הוא השירות שיריץ את האפליקציה שלך על האינטרנט. חינם.

1. היכנסי ל: **vercel.com**
2. לחצי **Sign Up** → בחרי **Continue with GitHub**
3. אשרי את החיבור (Authorize Vercel)
4. אחרי ההתחברות, לחצי **Add New... → Project**
5. תראי את `kai-chef-app` ברשימה - לחצי **Import** לצדו
6. במסך "Configure Project":
   - **Framework Preset:** Other (זה כבר יזוהה אוטומטית)
   - הרחיבי את **Environment Variables**
   - שם: `ANTHROPIC_API_KEY`
   - ערך: הדביקי את המפתח שיצרת בשלב 1 (`sk-ant-...`)
   - לחצי **Add**
7. לחצי **Deploy**
8. חכי ~1 דקה. תראי אנימציית קונפטי.
9. לחצי **Continue to Dashboard** ואז על שם הפרויקט
10. יש לך קישור! משהו כמו `kai-chef-app-xyz.vercel.app`

---

## שלב 5: בדיקה שהכל עובד

1. פתחי את הקישור בטלפון
2. וודאי:
   - האפליקציה נטענת
   - "שאל את השף" עונה תשובה (זה מוכיח שה-API עובד)
   - "מה יש במקרר?" מייצר מתכון
   - טיימרים במצב בישול פועלים
3. באייפון: Safari → כפתור השיתוף → "הוסף למסך הבית"
4. באנדרואיד: Chrome ישאל אותך לבד אם להתקין

**אם משהו לא עובד:** בדקי ב-Vercel → Deployments → Functions → Logs, שם תראי שגיאות.

---

## שלב 6: (אופציונלי) דומיין משלך

הקישור של Vercel עובד מצוין, אבל אם רוצה `kaichef.co.il`:

1. קני דומיין ב-**gandi.net** או **namecheap.com** (~40-100 ₪/שנה)
2. ב-Vercel: Project → Settings → Domains → Add
3. הזיני את הדומיין, ועקבי אחרי ההוראות (זה שינוי DNS פשוט)

---

## 🔒 אבטחה - למה שנוסיף בהמשך

כרגע כל אחד שיש לו את הקישור יכול לקרוא ל-API שלך. לפני שיש כמות משתמשים משמעותית, נוסיף:

1. **Rate limiting** - הגבלה של כמה בקשות ל-IP בדקה
2. **הזדהות אנונימית** - לוודא שרק המשתמשים באפליקציה יכולים לקרוא
3. **מעקב עלויות** - התראה במייל אם משתמש בזבזני

זה עוד ~30 דקות עבודה. נעשה את זה כשיהיו לך ~50 משתמשים.

---

## 📈 בדיקת שוק - איך לגשת נכון

1. **תני ל-30 אנשים ראשונים** (חברים, משפחה, קבוצות פייסבוק של בישול)
2. **ראייני 5-10** אחרי שבוע: מה השתמשת? מה חסר? האם תשלמי?
3. **מדדים לעקוב:**
   - כמה פתחו את האפליקציה יותר מפעם אחת?
   - כמה השתמשו בפיצ'ר של "מה יש במקרר"?
   - כמה סיימו מתכון ב-hands-free mode?

Vercel Analytics (חינם) יתן לך את המספרים הבסיסיים אוטומטית - הפעילי ב-Settings → Analytics.
